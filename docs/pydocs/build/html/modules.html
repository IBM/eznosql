pyeznosql.configure_logging(level=20, file_name='pyeznosql.log', log_format=None, delay=True)¶

Configure logging level, output file, and format for Python APIs

Parameters:

  • level (int, optional) – Python logging level, defaults to logging.INFO

  • file_name (str, optional) – the log output file name, defaults to ‘pyeznosql.log’

  • log_format (str, optional) – log formatted string, defaults to ‘%(asctime)s - %(name)s - %(levelname)s - [%(funcName)s] - %(message)s’

  • delay (bool, optional) – whether to delay opening the log file until the first log message is emitted, defaults to True
